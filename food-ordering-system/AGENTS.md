# Project Rules

This project is a microservices-based Online Food Ordering System.

Use:
- Java 17
- Spring Boot
- Maven
- H2 database
- Spring Data JPA
- Spring Web
- Validation
- Lombok
- springdoc-openapi-starter-webmvc-ui

Service rules:
- Use package format: com.foodordering.<service-name>
- Use layered structure: controller, dto, exception, model, repository, service
- Keep code clean and compile-ready
- Use file-based H2
- Add Swagger support
- Add data.sql sample data
- Follow consistent naming across all services
- Do not introduce unnecessary complexity
